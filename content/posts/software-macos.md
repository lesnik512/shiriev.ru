## Fresh install
1. Install [homebrew](https://brew.sh/)
2. Install xcode
```bash
xcode-select --install
```
3. Install [`Oh My Zsh`](https://ohmyz.sh/#install)
4. Install from brew:
```bash
brew install --cask iterm2 visual-studio-code appcleaner
```
